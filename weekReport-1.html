การเขียนสะท้อนตนเองแบบการเรียนรู้ผ่านประสบการณ์ Week # 1
(Recursive) ประสบการณ์ในสัปดาห์นี้มีการเชื่อมต่อกับประสบการณ์หรือแผนการที่ผ่านมาหรือไม่? มี ในช่วงต้นเดือนได้มีการฝึกงานด้านการพัฒนาเว็บไซต์จึงได้มีการใช้ React Framework ในการฝึกงาน ซึ่งมี html เกี่ยวข้อง
(CE) อะไรคือประสบการณ์ที่ผ่านมาของคุณ ? การเรียนรู้แท็กของภาษา html และ css
(RO) อะไรคือประสบการณ์ที่สำเร็จและล้มเหลว ?
      ประสบการณ์ที่สำเร็จคือ เขียนหน้าเว็บได้ด้วยตัวเอง
      ประสบการณ์ที่ล้มเหลวคือ เกิด Bug ที่โปรแกรม รายการจัดโครงสร้างของหน้าเว็บ ไม่ค่อยคล่อง
(AC) คุณได้เรียนรู้อะไรจากประสบการณ์ ? เรียนรู้จากความผิดพลาดที่ตัวเองเคยทำ และศึกษาตัวอย่างที่สำเร็จที่เคยมีมา
(AE) คุณจะได้นำสิ่งที่เรียนรู้จากประสบการณ์ ไปใช้อย่างไรต่อไป ? นำไปศึกษาต่อยอด เพื่อที่จะพัฒนาทักษะตัวเองเป็นพื้นฐานในการไปทำ FontEnd เพื่อเป็นเครื่องมือในการทำอาชีพ
